Fit to iPhone

16.06.2020

So, with the new design I also wanted to test some "features" of iOS where one could imitate the feel of a native app. The idea is to place a shortcut to the website on the homescreen and have a nice icon. When one launches this website-shortcut it should open in a Safari window without address-bar and properly colored statusbar. Appscope has a short article on how to do that.


For the new, second design I did not use a splashscreen, though I did add them by now (also using the splashscreen generator linked in the article above).

After I fiddled out how this has to be done I was quite happy that it looked how I expected it. But after looking onto it again I found one thing I really didn't like: the logo in the top left corner. While I like the logo itself, even though it's very simple, I don't like how it looks in this "native-like" website-shortcut.

When turning the phone to landscape everything is fine, the status bar of the phone disappears and the design does not break. But in portrait mode, the mode I use most, it looks horrible to have this little extra bar above the logo.


This annoyed me so much, that I had to remove the logo and replace it by something different - the rainbow color line - immediately. You can see the result. The result looks somewhat boring. I kept the colors, but something is missing now.


Somehow and somewhen the logo will return!

Development HTTP Server

26.12.2019

For developing this website I needed a simple way to test it. Since it is just a plain HTML file with some CSS I could just open the file in the browser. But to see how it looks on the iPhone (besides the responsive mode of the browser) I needed running server the phone could connect to.


A quick google took me to MDN. The article shows that Python3's built in HTTP Server is just enough. Having Pyhton already installed serving the website to the iPhone was quick.

Beginning Layout

25.12.2019

This text and the HTML and CSS it is embedded in and represented by marks the beginning of my path of learning these technologies and specifically aspects of design and layout.
